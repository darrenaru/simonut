# MOTARA -- Sistem Informasi Instalasi Farmasi

Website ini merupakan dashboard untuk membantu instalasi farmasi dalam
mengelola data obat dan aktivitas terkait. Sistem ini menyediakan
antarmuka sederhana untuk menampilkan menu utama, melakukan pencatatan
transaksi obat, serta menampilkan informasi penting terkait stok dan
aktivitas pengguna.

## Fitur Utama

-   **Dashboard Admin** untuk navigasi cepat ke modul utama.
-   **Pencatatan Obat Keluar & Masuk** melalui halaman transaksi.
-   **Pengelolaan Pengguna dan Akses Role** menggunakan skrip
    JavaScript.
-   **Integrasi Notifikasi** pada navbar.
-   **Desain Responsif** menggunakan CSS custom tanpa framework.

## Teknologi

-   HTML
-   CSS
-   JavaScript
